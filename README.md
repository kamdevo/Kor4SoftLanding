# Landing Page Kor4Soft

Landing page moderna creada con Next.js, React y Tailwind CSS, inspirada en un diseño limpio y profesional.

## Tecnologías

- **Next.js 14** - Framework de React
- **React 18** - Biblioteca de UI
- **Tailwind CSS** - Framework de CSS utility-first
- **TypeScript** - Tipado estático

## Instalación

1. Instala las dependencias:
```bash
npm install
```

2. Ejecuta el servidor de desarrollo:
```bash
npm run dev
```

3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura del Proyecto

```
├── app/
│   ├── layout.tsx      # Layout principal
│   ├── page.tsx        # Página principal (landing)
│   └── globals.css     # Estilos globales con Tailwind
├── components/
│   ├── Header.tsx              # Header con logo y navegación
│   ├── Hero.tsx                # Sección hero principal
│   ├── EarningsCard.tsx        # Card de ganancias
│   ├── ConnectSourcesCard.tsx  # Card de conexión de fuentes
│   └── LearnMoreCard.tsx       # Card de aprender más
└── public/
    └── Logo.png        # Logo de Kor4Soft
```
